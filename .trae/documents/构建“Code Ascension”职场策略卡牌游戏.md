# Code Ascension: 互联网创业生存指南

我将为你构建一款基于 Web 的单人策略卡牌游戏，主题围绕“互联网创业与职场生存”。玩家将扮演一名创业者/程序员，通过打出各种“职场技能卡牌”来应对“项目危机”和“技术债务”。

## 游戏设计概要
*   **核心机制**：回合制战斗。玩家通过消耗“精力(Energy)”打出卡牌，对敌人造成伤害（推进进度）或防御（解决 Bug/公关）。
*   **现实关联**：
    *   **HP (生命值)** -> **资金 (Funding)**：归零即破产（Game Over）。
    *   **MP (法力值)** -> **精力 (Energy)**：每回合刷新，用于打牌。
    *   **护甲 (Armor)** -> **稳定性 (Stability)**：抵挡敌人的攻击（Bug/压力）。
    *   **敌人 (Enemies)** -> **项目危机**：如“需求蔓延”、“服务器宕机”、“挑剔的甲方”。
*   **策略深度**：
    *   **Debuff 管理**：敌人会向你的牌库塞入“Bug”卡（抽到不仅没用还扣血），你需要用“重构”卡来移除它们。
    *   **透支机制**：使用“加班”卡可以获得短期爆发，但会导致下回合“过劳”（精力减少）。

## 技术实现计划

### Phase 1: 项目初始化
1.  使用 `Vite` + `React` + `TypeScript` 搭建高性能 Web 应用框架。
2.  配置 `Tailwind CSS` 用于快速构建现代化的 UI 界面。

### Phase 2: 核心引擎开发
1.  **数据结构**：定义 `Card` (卡牌), `Player` (玩家), `Enemy` (敌人) 的 TypeScript 接口。
2.  **战斗系统**：实现回合循环、抽牌逻辑、洗牌逻辑、伤害结算系统。
3.  **状态管理**：构建一个轻量级的游戏状态机（Store），管理战场上的所有数据。

### Phase 3: 内容与 UI 实现
1.  **卡牌设计**：
    *   **攻击牌**：`写代码` (基础攻击), `全栈开发` (高伤), `快速迭代` (连击)。
    *   **技能牌**：`咖啡续命` (回蓝), `画大饼` (获得临时护盾), `外包` (高费高伤)。
    *   **诅咒牌**：`Bug`, `技术债`, `无效会议`。
2.  **敌人设计**：
    *   `Legacy Code (遗留代码)`：血厚，不断产出 Bug。
    *   `Deadline (截止日期)`：每回合攻击力倍增，必须速战速决。
3.  **UI 组件**：
    *   制作精美的卡牌组件（包含卡名、消耗、描述、插图位）。
    *   带有动画效果的战场布局。

### Phase 4: 优化与平衡
1.  添加简单的 CSS 动画（抽牌、攻击特效）。
2.  调整卡牌数值，确保游戏既有挑战性又不会太难上手。

待确认后，我将立即开始代码实现。
